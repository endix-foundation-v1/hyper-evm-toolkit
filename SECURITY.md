# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| `main`  | Yes |
| Older tags | No |

## Reporting a vulnerability

Do not report vulnerabilities through public GitHub issues.

Use GitHub Private Vulnerability Reporting:

- https://github.com/endix-foundation-v1/hyper-evm-toolkit/security/advisories/new

Please include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Suggested mitigation (optional)

## Response policy

- Initial acknowledgement target: within 48 hours
- We will investigate, prioritize, and coordinate disclosure before public release.
